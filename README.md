# 🎬 Let'sFioHub - Let's-1.0

**Мировой видеохостинг с геймерскими темами**

## 🌟 Возможности
- 🎮 9 геймерских тем (Cyberpunk, CoD, Fortnite, Minecraft, Valorant, LoL, Dark Souls, Apex)
- 📺 Загрузка и просмотр видео
- 💬 Комментарии и лайки
- 👥 Каналы и подписки
- 💰 Монетизация
- 📱 Fast - короткие видео

## 🛠️ Технологии
- **Backend:** Python FastAPI + SQLite
- **Frontend:** React Next.js
- **База данных:** SQLite
